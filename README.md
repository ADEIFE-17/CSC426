**CSC426**

This repository contains multiple frontend development projects built using **HTML, CSS, and JavaScript** as part of my CSC426 practice.

**1. SIMPLE CALCULATOR**
This is a user-friendly calculator.
The calculator performs basic arithmetic operations through a clean graphical user interface.

**FEATURES**

- Addition (+)
- Subtraction (-)
- Multiplication (\*)
- Division (/)
- Exponent (^)
- Modulus (%)
- Clear (C)
- Decimal calculations
- Error handling for invalid expressions
- Responsive and modern design

**2. LOG-IN AUTHENTICATION SYSTEM**

This is a simple web-based authentication system.  
The application demonstrates a full user flow including **signup, login, validation, and success feedback screens**.

**FEATURES**

User Signup (Create Account)

- User Login Authentication
- Input Validation (empty fields, invalid credentials, duplicate users)
- Success and Error Messages
- Reset / Cancel Buttons
- Responsive User Interface
- Page flow between Signup → Login → Success screen
- LocalStorage used to store user data (frontend simulation)

**PROJECT FLOW**

1. User creates an account (Signup page)
2. System validates input and stores user data
3. User is redirected to Login page
4. User logs in with correct credentials
5. Success message is displayed
6. User can return to login using Cancel button

**TECHNOLOGIES USED**

HTML5
CSS3
JavaScript (ES6)
